# Real Estate Intelligence Dashboard RD

Proyecto profesional de scraping inmobiliario + dashboard interactivo con Python y Streamlit.

## Tecnologías
- Python
- Pandas
- BeautifulSoup
- Streamlit
- Plotly

## Funcionalidades
- Extracción de propiedades
- Limpieza automática de datos
- Dashboard interactivo
- KPIs inmobiliarios
- Filtros dinámicos
- Exportación CSV

## Instalación

```bash
pip install -r requirements.txt
```

## Ejecutar scraper

```bash
python scraper/scraper.py
```

## Ejecutar dashboard

```bash
streamlit run dashboard/app.py
```

## Deployment
Recomendado:
Streamlit Community Cloud
